# BookMyShow UX Case Study

## Overview

This project presents a comprehensive UX case study of the BookMyShow mobile application. The study evaluates the existing user experience through user research techniques, journey mapping, and heuristic evaluation. The objective is to identify usability challenges and propose practical design solutions that enhance the overall ticket booking experience.

## Project Objectives

* Analyze the current user experience of BookMyShow.
* Identify usability strengths and pain points.
* Apply Jakob Nielsen's Usability Heuristics.
* Understand user behavior through persona creation and journey mapping.
* Propose UX/UI improvements to improve efficiency, usability, and user satisfaction.

## Case Study Process

### 1. Executive Summary & Overview

An overview of BookMyShow, its purpose, target audience, and key user experience goals.

### 2. User Persona Definition

Creation of a representative user persona to understand user needs, behaviors, goals, and pain points.

### 3. User Journey Mapping

Analysis of the end-to-end booking journey, including:

* App Launch & Discovery
* Movie Selection
* Theatre & Showtime Selection
* Seat Selection
* Payment & Checkout
* Booking Confirmation

### 4. Heuristic Evaluation

Evaluation based on Jakob Nielsen's 10 Usability Heuristics to identify strengths and opportunities for improvement.

### 5. Proposed UX/UI Interventions

Three strategic design improvements were proposed:

#### Transparent Pricing System

Improves pricing visibility by displaying all charges earlier in the booking process.

#### Simplified Home Screen Navigation

Reduces information overload and prioritizes core booking actions.

#### Smart Seat Recommendation System

Helps users quickly identify the best available seats through intelligent recommendations.

### 6. Conclusion

Summarizes key findings and highlights opportunities to improve user satisfaction, booking efficiency, and platform usability.

## Tools Used

## Key Outcomes

* Improved pricing transparency
* Reduced cognitive load
* Enhanced navigation efficiency
* Faster seat selection process
* Better overall user experience
